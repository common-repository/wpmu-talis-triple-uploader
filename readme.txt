== Description ==

This plugin uploads RDF Triples to be uploaded to the Talis Platform.

The RDF Triples are generated by Triplify (http://triplify.org/) with this configuration file http://code.google.com/p/jiscpress/source/browse/#svn/trunk/WPMU%20Triplify%20Config

This plugin requires a Talis Platform account from http://www.talis.com/

This plugin was created for the JISC (http://jisc.ac.uk) funded JISCPress project (http://jiscpress.blogs.lincoln.ac.uk)

== Installation ==

1. Place this plugin in your /wp-content/plugins/ folder.
2. Activate the plugin sitewide
3. In the Site Admin panel, click "Talis Global Options" to configure.
4. Each blog that wishes to upload it's triples needs to be set to publicly accessible and agree to the terms and conditions.

== Frequently Asked Questions ==

= Will this plugin submit changes to Talis? =

No it just uploads entire Triples each time.
